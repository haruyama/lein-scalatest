# lein-scalatest

A plugin to access the scalatest testing framework from Leiningen.

## Usage

Put `[haruyama/lein-scalatest "0.0.4"]` into `:plugins`.  Requires Leiningen version 2.0.0 or greater.

To run, simply execute:

    $ lein scalatest
